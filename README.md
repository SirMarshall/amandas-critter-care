# 🐾 Amanda's Critter Care

**Expert Care When You Can't Be There.**

Professional, loving pet sitting and dog walking services tailored to your furry family members across the Space Coast and Brevard County, FL.

## ✨ Features

- **Dog Walking:** Active, engaging walks tailored to your dog's energy level.
- **Overnight Stays:** Premium in-home care so your pets can sleep in their own beds.
- **Cat Sitting:** Drop-in visits for feeding, playtime, and companionship.
- **Trust & Safety:** Fully insured & bonded, Pet CPR certified, and 5-star rated on Google.

## 🛠️ Technology Stack

This website is built using modern web technologies:

- **React** - Frontend library for building interactive UIs.
- **Vite** - High-performance build tool and development server.
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development.
- **Lucide React** - Beautifully simple pixel-perfect icons.
- **Framer Motion** - Production-ready motion library for animations.


## 📍 Service Areas

Proudly serving the Space Coast including Melbourne, Viera, Cocoa Beach, Rockledge, and Merritt Island.
